Directory Structure to Text

10 Aug 2022 Tools documentation tool 3 min read

This is a tool where you can select a file on your PC and it will be rendered into text to then be used for documentation. This is super useful for annotating project file structures.

Exclude Filter 
Output will appear here after selecting a directory.

Example Filters

These are some example filters for specific project types

Python

Ignore .git, .venv, __pycache__, .idea

(^|\/)(?:\.git|.venv|__pycache__|.idea)(?:\/|$)

C#

Ignore .git, .vs, bin, obj

(^|\/)(?:\.git|.vs|bin|obj)(?:\/|$)

Node/JavaScript

Ignore .git, node_modules, build, dist

(^|\/)(?:\.git|node_modules|build|dist)(?:\/|$)
